1.one.

Reel Stars Student Film Festival

Poster | DVD | DVD Cover »

I was asked to create a poster, DVD, and DVD cover for the Reel Stars Student Film Festival. Certain elements were required like including the sponsors. I decided that I wanted to do something on a dark background with a lot of colours. At first it was top heavy but after a little re-working, it was perfect. I really wanted a lot of colours to offset the black and white and that is why I love the filmstrip in the middle.

2.TWO.

Burst Gum.

Advertising Designs | NEW »

I was asked to create a gum package for a new gum company called "Burst". I decided to go with another black theme but highlight it with blue and yellow. I thought it would be nice to have a really cool space/fantasy type feel to the gum while still feeling professional.

3.Tri.

Red Space

Photorealism | Spaceships »

I was asked to create a 30 second clip using photos and video clips. I composited together this idea of a red space. I thought it would be really cool and I loved how the background turned out as it moved.

DOTA Music

Defence of the Ancients | Audio »

I was asked to create a 3 minute song for DOTA (Defence of the Ancients), a mini-game in World of Warcraft 3: The Frozen Throne. The mashup was a lot of fun!


5.FiVe.

Actionscript Extravaganza

Flash | play »

For this project I was asked to complete a numerous amount of tasks involving flash but with minimal use of tweens, drawing, and so forth. Instead this project consisted of mostly action script and being able to code different symbols to interact with eachother in order to get a desired affect. This project took a lot of creativity and in the end the result was really great!


6.SIX.

Asteroids-X

Flash | games | GO »

In this flash project I was asked to create an old school ateroids style arcade game. I decided to infuse a lot of colour in this project and to create a few cool additions like being able to move the ship in the start screen. This project required a lot of attention to detail and was one of the more advance projects that I have worked on.
